# Ëtrid Documentation

Official documentation for the Ëtrid Network. Technical specifications, API references, validator guides, developer tutorials, and governance documentation. Covers Primearc Core Chain, Partition Burst Chains, ASF consensus, PrimeSwap DEX, and cross-chain bridge operations.

## Repository

**Push to:** `git@github.com:etaborai/etrid-docs.git`

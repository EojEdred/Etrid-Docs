# Etrid Documentation

Welcome to the official documentation for the Etrid Network.

## What is Etrid?

Etrid is a next-generation multichain blockchain platform featuring:

- **Primearc Core Chain** - The main relay chain using ASF consensus
- **Partition Burst Chains (PBCs)** - 14 specialized chains for different assets
- **Lightning Bloc** - Instant payment channels
- **Cross-chain Bridges** - Trustless asset transfers
